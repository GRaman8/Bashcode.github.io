Bash Commands Guide

cd
Usage: Change directory
Other Ways: chdir (less common)
Description: Navigates to a different directory.
ls
Usage: List directory contents
Other Ways: dir (on some systems)
Description: Lists files and directories in the current directory.div>
pwd
Usage: Print Working Directory
Other Ways: None
Description: Displays the full path of the current working directory.
mkdir
Usage: Create a new directory
Other Ways: None
Description: Creates a new directory.
rmdir
Usage: Remove an empty directory
Other Ways: rm -d (be cautious!)
Description: Removes an empty directory. Use rm -d with caution as it can permanently delete non-empty directories.
touch
Usage:Create an empty file
Other Ways: None
Description: Creates an empty file.
rm
Usage: Remove a file or directory (use with caution!)
Other Ways: del (on some systems)
Description: Removes a file or directory. Use with caution!
cp
Usage: Copy a file or directory
Other Ways: None
Description: Copies a file or directory.
mv
Usage: Move or rename a file or directory
Other Ways: ren (on some systems)
Description: Moves or renames a file or directory.
cat
Usage: Display the contents of a file
Other Ways: None
Description: Displays the entire content of a file.
more
Usage: Display a file content page by page
Other Ways: less (similar functionality)
Description: Displays the content of a file one page at a time.
grep
Usage: Search for a pattern in text files
Other Ways: findstr (on some systems)
Description: Searches for lines matching a specific pattern in text files.
echo
Usage: Print text to the terminal
Other Ways: printf (more formatting options)
Description: Prints text to the command line.
chmod
Usage: Change file permissions
Other Ways: None
Description: Modifies access permissions (read, write, execute) for files and directories.
head
Usage: Display the first lines of a file
Other Ways: None
Description: Displays the first n lines (default: 10) of a file.
tail
Usage: Display the last lines of a file
Other Ways: None
Description: Displays the last n lines (default: 10) of a file.
**
Usage: ` (pipe operator)**
Other Ways: Connects commands
Description: It matches filenames and directories recursively.
wc
Usage: Word count
Other Ways: None
Description: Counts lines, words, and characters in a file.
sed
Usage: Stream editor
Other Ways: None
Description: A powerful text processing tool for filtering, modifying, and transforming text files.
awk
Usage: Pattern-matching language
Other Ways: None
Description: Another powerful text processing tool for data manipulation based on patterns and conditions.
